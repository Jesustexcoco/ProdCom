FloorComm One-time setup · 5 min · free forever STEP 1 — CREATE FIREBASE PROJECT Go to Firebase (free, uses your Google account) 1. Open console.firebase.google.com 2. Click "Add project" → name it floorcomm → Continue 3. Turn OFF Google Analytics → click "Create project" 4. Wait → click "Continue" 💡 Same Google account as Gmail — no new signup needed STEP 2 — ENABLE REALTIME DATABASE Turn on the database so devices can talk to each other Inside your Firebase project: 1. In the left menu click "Build" → "Realtime Database" 2. Click "Create database" 3. Pick any location → click Next 4. Select "Start in test mode" → click "Enable" STEP 3 — GET YOUR APP KEYS Register a web app and copy the config 1. Click the ⚙ gear icon → "Project settings" 2. Scroll to "Your apps" → click the Web icon 3. Name it floorcomm → click "Register app" 4. Copy the values from the firebaseConfig block into the fields below For the databaseURL — go back to Realtime Database, it shows at the top like: https://floorcomm-xxxxx-default-rtdb.firebaseio.com Connect & Launch App → Try demo mode (no real sync) ALREADY SET UP?
